# OrangeHRM Playwright Automation Framework

## Project Overview

This project is an end-to-end UI Automation Framework developed using Playwright with Java.

## Technology Stack

- Java
- Playwright
- TestNG
- Maven
- Page Object Model (POM)
- Git

## Features

- Login Automation
- Add Employee
- Search Employee
- Logout
- Cross Browser Testing
- Parallel Execution
- Screenshot on Failure

## Framework Structure

```
src
 ├── pages
 ├── tests
 ├── base
 ├── utilities
 ├── config
 └── resources
```

## Browser Support

- Chromium
- Firefox
- WebKit

## How to Execute

```
mvn clean test
```
